# misskey follow import
## about this
API使ってMisskeyのフォローインポートできるやつ。    
フォローインポートが正常に機能しないインスタンスの住民に向けて。

## usage
```
$ npm i
$ node index.js SERVER_DOMAIN API_TOKEN CSV_FILE_PATH
```
